    NYSETL.Extra.Scrubber

    Functions to scrub all PII in ECLRS dump files so that it can be used in development. Sensitive data is replaced with random strings of the same length and format (letters for letters, digits for digits and dates for dates). Currently only works with the 45-column ECLRS files but can easily be extended.
